The key thing to note is that Mac and Windows are designed differently. Mac uses forward slashes to indicate a folder change and Windows uses backslashes. But their structures are different as well. Notice that Windows has “C:” at the beginning. This is the C drive and there are other drives as well. Other differences were introduced in [provisional cross-reference: fm_differences] but the details are beyond the scope of this book (and likely beyond my understanding). A brief history on file paths is given by How-To Geek 23  and a rather hard-to-read explanation can be found on Wikipedia 24 .
